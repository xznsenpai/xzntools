#### Install script on Termux
```bash
$ pkg update && pkg upgrade
$ pkg install python
$ pkg install git
$ pip install requests
$ pip install bs4
$ git clone https://github.com/Xenzi-XN1/YT-Download
```
#### Run script
```bash
$ cd YT-Download
$ pip install-r modul.txt
$ python main.py
```
