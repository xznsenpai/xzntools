#### Install script on Termux
```bash
$ pkg update && pkg upgrade
$ pkg install nodejs
$ pkg install git
$ git clone https://github.com/xznsenpai/xzntools
```
#### Run script
```bash
$ cd xzntools
$ npm install
$ npm start
```
